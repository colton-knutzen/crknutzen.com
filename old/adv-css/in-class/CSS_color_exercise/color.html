Let's play with some color!

This is page content for color.

Color can be specified for web pages by using RGB values, hexdecimal values, HSB values, and color names!

Opacity (AKA transparency) can be applied to anything with color. This includes text color, background color, and images.

Use this page and CSS styesheet to practice with color and opacity values. Use the various resources provided on blackboard to change the colors in the text and background of the elements on this page.

Use the CSS opacity property to change the opacity of the image.

Create a gradient in the background of the container.

Use the rgba specification to change the transparency of oa color and/or a dropshadow.
