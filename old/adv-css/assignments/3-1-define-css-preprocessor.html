3-1: Define CSS Preprocessor

What does a CSS Preprocessor do?

A CSS preprocessor takes their own syntax, which is usually simplier or more efficient, and converts it into CSS document that can be used by the browser.

What are the advantages of using a CSS Preprocessor?

They allow you to reduce some of the clutter that traditional CSS has. Allows you to write "less" code, saving time and being more effcient. And easier to make changes and troubleshoot since there will be less code to shift through and more universal variables.

What are the disadvantages or issues you need to consider when using a CSS Preprocessor?

Like with adding any new tool, there could be issues with the translation, or the coder is unable to write it in a way that the Preprocessor is able to covert it to CSS in a way that the developer intends.

List at least Three CSS Preprocessors currently available.

Sass, LESS, Stylus
